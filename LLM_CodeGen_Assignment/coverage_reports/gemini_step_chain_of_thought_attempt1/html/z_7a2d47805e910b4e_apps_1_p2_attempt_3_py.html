7def shortest_path_with_obstacles(grid: List[List[int]], k: int) -> int: 

8 ''' 

9 You are given a grid of size N x M where: 

10 - 0 represents an empty cell 

11 - 1 represents an obstacle 

12 - 2 represents the start position 

13 - 3 represents the end position 

14  

15 You can move in 4 directions (up, down, left, right). 

16 However, you have a special ability: you can remove at most K obstacles during your journey. 

17  

18 Find the shortest path from start to end, considering you can remove at most K obstacles. 

19 Return the minimum number of steps, or -1 if impossible. 

20  

21 Parameters: 

22 - grid: 2D list representing the grid 

23 - k: maximum number of obstacles you can remove 

24  

25 Example: 

26 grid = [[2, 1, 0, 0], 

27 [0, 1, 1, 0],  

28 [0, 0, 1, 3]] 

29 k = 1 

30  

31 Without removing obstacles: impossible 

32 With removing 1 obstacle: can reach in 5 steps 

33 '''

41 for c in range(cols):

52 # The state includes the number of obstacles removed because reaching the same cell

57 

58 while queue:

60

63 return steps
